## moonbit 基本语法
1. moonbit 是面向值的. 通过获取值来实现程序的运行
  - 不同的值有不同的类型，对应了不同的运算符
  - moonbit 的表达式实际上可以看做值的定义
2. moonbi是静态类型。 也就是在编译前就确定了类型是否正确

## 表达式块
```js
{
  let a:int = 3
  let b:int = 4
  5
}
```
这个表达式块的类型就是最后的表达式的类型 int,值就是 5
我们可以加上 if 关键字来判断选用的表达式值(类似于三元运算符)
```js
(if 1 < 100 {1} else {0})*100
```
## 类型转换
